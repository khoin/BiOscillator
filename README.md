BiOscillator
===========

![screenshot](http://i.imgur.com/WaLvXbZ.png)

## Polyphonic 2-OSC Synthesizer

### Features

* 2 Oscillators
* Amplitude / Filter Envelopes
* Pitchbend Supported
* Vibrato

### Planned
* Preset Library
* Sequencer/Arpeggiator
